# HireNext User Guide

## Job Adder Guide

1. Login with a Job Adder account.
2. Open recruiter dashboard.
3. Create a job:
   - Fill company, role, location, description.
   - Select `Open` or `Restricted` access mode.
4. For restricted jobs:
   - Select recruiters in multi-select.
   - Save with optional notes.
5. Manage access:
   - Open `Jobs Management`.
   - Click `Edit Access`.
   - Add/remove recruiters and save.
6. View recruiter performance:
   - Open `Recruiter Performance` tab.
   - Use sorting and search.

## Recruiter Guide

1. Login with recruiter credentials.
2. Open `Available Jobs`.
3. Use filters (location/company/search).
4. Click `Submit Resume` on a job.
5. Fill candidate form and upload file (`pdf/doc/docx`, max `5MB`).
6. Submit and verify success message.
7. Check dashboard metrics for updated `submitted` count.

## Admin Guide (Current + Planned)

Current:
1. Login through admin login page.
2. Open admin dashboard modules (revenue, candidate submissions, top resumes).

Planned (Phase 2):
1. Manual status counter updates (`verified`, `select`, `joined`).
2. Bulk counter updates.
